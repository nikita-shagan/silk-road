# Шелковый путь

Сайт международного ралли со сложными маршрутами по пересеченной местности.

#### _Технологии, используемые в проекте:_

*   Grid layout
*   Flexbox
*   Методология BEM
*   Адаптивная верстка (медиа запросы)
*   "Резиновая" верстка
